Proyecyo Java 1: Juego de Ajedrez


Mi primer "gran" proyecto en Java esta siendo un juego de ajedrez. Para un aprendiz de este mundo como yo esta siendo un gran desafio, principalmente por el tiempo que le puedo dedicar, pero es una maravillosa forma de practicar y afianzar conocimientos, para asimilar y perfectionar la prograamción orientada a objetos.
El modo de juego sera para dos jugadores, sin poder jugar contra una CPU.

Proyecyo Java 2: Juego del ahoracado


Proyecto que estoy desarrollando a la vez que el ajedrez, al tratarse de un juego mas sencillo, el numero de clases es menor, por lo que aunque lo empece a posteriori de la ajedrez, va a ser el primer juego terminado al 100%.
Proximamente, al igual que con el juego de ajedrez, subire el desarrollo de esta juego a su pagina web correspondiente, con una versión descargable para poder probarlo y asi recibir feedback de su funcionamiento.
